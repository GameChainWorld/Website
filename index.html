GameChain is a smart contract cross-chain Gaming platform that allows game developers to create crowdsales for games that they develop on the GameChain platform. The GameChain ecosystem allows Game Developers and Gamers alike to monetize content, utilize masternodes, play 'click to play games' and earn tokens from interacting within the community.
GameChains unique reward system creates an economic incentive similar to Steemit for users to create content on the GameChain platform. This ensures longevity and growth of the GameChain ecosystem. Unlike other platforms (Steam, Origin, GOG, etc.) GameChain provides game developers with smart contract and game development tools, and the ability to create crowdsales raising funds from a currency type of the game developers choosing and using atomic swaps convert the funds to a stable currency.

Fair and transparent economic community incentives
Players, developers, investors, critics and promoters can all interact within the community. Interactions are executed by smart contracts which will write the event to the blockchain in exchange creating a reputation for the user. Community interactions are rewarded based on the interaction performed and reputation of user similar to Steemit’s community.
Easily add decentralized items, games, and currencies to any game
Allow in-game items and interactions to go beyond the game. Using GameChain’s SDK game developers can easily create decentralized in-game items, incorporate any cryptocurrency into the game using atomic swaps, and allow the games themselves to become tradable digital assets that can be exchanged between players
Reputation based platform
Reward quality community members and control click farming through GameChain’s player reputation system. Measure a users reputation by previous game scores, quality of content produced, and by participating in the platform. Users reputation will help game investors and players make their decisions in which games to fund. Reputation also has an impact on the amount of rewards received from community interaction
Blockchain agnostic and cross-chain atomic swaps
GameChain is blockchain agnostic and allows game developers to utilize any digital currency in both their game development crowdsale and in-game purchases.
Crowdsale fund volatility solved
Game Developers can create crowdsales to fund their game development. Funds that are collected during the crowdsale are converted using atomic swaps to a stable currency. This eliminates any price volatility that plague cryptocurrency crowdsales.
Game Development Software Engine
Use GameChain’s software Bcreator.io to create click to play video games quickly with ease. Bcreator.io can be used with a graphical interface, or the SDK can be accessed using C#, JavaScript and other languages.
Smart Contracts Graphical Programming-free Development Tool
Build smart contracts for game development using intuitive drag-and-drop interfaces. Anyone with inspiration and creativity can create games with GameChain.
Click it to play HTML5 2.0
Easily convert existing high quality native games to ‘click to play’ versions that are mobile and web friendly. No downloading required.
